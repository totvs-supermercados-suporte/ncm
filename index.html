Documentação - Consulta NCM

          
        

        
        
        

          
          
Introdução

          
O Consulta NCM é uma ferramenta desenvolvida para facilitar a validação e consulta de NCMs (Nomenclatura Comum do Mercosul) em lotes. Com esta ferramenta, você pode importar um arquivo XML contendo uma lista de NCMs e verificar sua validade. Além disso, você pode consultar individualmente um NCM inválido para obter mais informações e garantir o cumprimento das regulamentações fiscais.


          
Como usar

          
    
            
  1. Na seção "Importe seu arquivo XML abaixo", clique no botão "Selecionar arquivo" e escolha o arquivo XML que contém os NCMs que você deseja validar.
    2. 
            
  2. Clique no botão "Processar" para iniciar o processo de validação.
    3. 
            
  3. Após o processamento, a seção "NCMs encontrados" exibirá a lista de NCMs contidos no arquivo XML e sua validade.
    4. 
            
  4. Para consultar individualmente um NCM inválido, vá para a seção "Consulte seu NCM inválido" e insira o NCM desejado. Uma descrição detalhada será exibida, fornecendo informações úteis sobre o NCM.
    5. 
            
  5. Observação: É importante ressaltar que o número de rejeição de NCM no ERP é o número referente ao XML e não ao item cadastrado no sistema. Por isso, importe o arquivo XML aqui e verifique o número de rejeição nos sistema.
    6.
